A creative WordPress blog/magazine theme

== Description ==

Canos is a creative WordPress blog/magazine theme built for blogs, newspapers, or magazines. Developed with SEO best practices in mind (and microdata optimized), Canos lets you to write articles and blog posts easily and quickly.

== Installation ==

Upload the **canos.zip** file in 'Appearance > Themes > Add New > Upload' or upload via FTP the extracted **canos** folder in 'wp-content/themes/'.
